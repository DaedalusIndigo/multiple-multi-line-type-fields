# Multiple Multi-Line Type Fields
An Anki add-on that creates a development framework for creating and displaying new input kinds, whereas Anki natively supports one single-line input per card.

As Multiple Multi-Line Type Fields (MMTF) is new, it is currently unavailable on [AnkiWeb](https://ankiweb.net/shared/addons).
